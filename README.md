# 🔬 HITRAN Simulator (2025-07-09)

**최신 업데이트: 2025-07-09**

---

## 📢 소개

HITRAN Simulator는 HITRAN 데이터베이스 기반의 분자 스펙트럼 시뮬레이션, 실험 데이터 비교, 고급 분석(파라미터 스윕, 라인 분석 등)을 지원하는 오픈소스 웹 애플리케이션입니다. 실험실, 연구, 교육, 논문 작성에 최적화되어 있습니다.

---

## 🚀 주요 기능

- **혼합/농도별 분석**: 여러 분자 동시 혼합, 농도 스윕, OA-ICOS 등 실험적 분석
- **실험 데이터 업로드/오버레이**: CSV/TXT 업로드, 시뮬레이션과 실험 데이터 직접 비교, 통계 분석(상관계수, RMS 등)
- **고급 시각화**: 2D/3D 스펙트럼, 피크 자동 감지, 라인 정보 툴팁, 라인 통계, 분포, 피팅, 강도 비교, 온도/압력 스윕(히트맵/3D)
- **파라미터 스윕**: 온도/압력/농도별 반복 계산, 최적 조건 탐색, 히트맵/3D 시각화
- **다국어 지원**: 한국어/영어/일본어/중국어 UI
- **결과 내보내기**: CSV, PNG(그래프), PDF(예정)
- **실험기기 파라미터 반영**: 미러 반사율, 캐비티 길이, 노이즈 등
- **모듈화/확장성**: 새로운 기능 추가 및 커스터마이즈 용이

## 🖥️ 설치 및 실행

### 1. 환경 준비
- Python 3.9 이상 권장
- (선택) 가상환경 권장: `python -m venv hitran_env && source hitran_env/bin/activate`

### 2. 의존성 설치
```bash
pip install -r requirements.txt
```

### 3. 실행
```bash
streamlit run ui/HITRAN_Simulator.py
```
- 기본 포트: 8501 (이미 사용 중이면 `--server.port 8502` 등 지정)
- 웹브라우저에서 `http://localhost:8501` 접속

```csv
wavelength,absorbance

1390.1,0.0013
...
```
- **파장 단위**: nm
- **흡광도/투과도/투과율** 모두 지원 (업로드 시 선택)

## 🧑‍🔬 사용법 요약
1. **사이드바에서 분석 조건 설정** (분자, 온도, 압력, 경로, 해상도 등)
2. **혼합/농도별 분석 모드 선택**
3. **[실험 데이터] 탭에서 CSV 업로드, 오버레이/통계 확인**
4. **[라인 분석] 탭에서 피크, 라인 통계, 피팅 등 고급 분석**
5. **[파라미터 스윕] 탭에서 온도/압력 스윕, 최적 조건 탐색**
6. **[내보내기] 탭에서 CSV/PNG 다운로드**

---

## 🌡️ 파라미터 스윕 예시
- 온도: 200~400K, 10단계 → 온도별 스펙트럼/최대 흡광도/피크 위치/민감도 자동 분석
- 압력: 100~1500 torr, 10단계 → 압력별 분석
- 온도+압력: 2D 히트맵/3D 서피스, 최적 조건 자동 탐색

## ❓ FAQ
- **Q. 파라미터 스윕이 안 보입니다.**
  - A. "농도별 분석" 모드에서 한 분자만 선택해야 활성화됩니다.
- **Q. 포트 충돌 오류가 납니다.**
  - A. `--server.port 8502` 등 다른 포트로 실행하세요.
- **Q. 실험 데이터와 단위가 다릅니다.**
  - A. 업로드 시 데이터 타입(흡광도/투과도/투과율) 선택 가능, 자동 변환됩니다.
- **Q. 기능 추가/수정 요청은?**
  - A. 이슈 등록 또는 직접 Pull Request 환영!
